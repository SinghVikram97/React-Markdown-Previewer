# Markdown Previewer

### Markdown Previewer built using

- [React](https://reactjs.org/)
- [Firebase](https://firebase.google.com/)
- [react-markdown](https://www.npmjs.com/package/react-markdown)

### To start

- Clone the repository
- `npm install`
- `npm start`
- Open http://localhost:3000
